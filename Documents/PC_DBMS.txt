Point cloud visualization/ query system using database backend.

This document describes the alternative system for point cloud visualization and querying using point clouds and their semantics stored in a database system.

Steps:
1) create a database system storing the point cloud representation created by the POTree converter;

2) Set up a web-server for dynamic content. The server receives the URL requests from the POTree viewer and modifies them into database queries to extract the requested .js or .laz files, and returns the responce to the viewer.
